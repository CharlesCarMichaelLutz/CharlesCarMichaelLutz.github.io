Paulie Social

The Paulie Social application implements an interactive React JS client where visitors can request tweets of their favorite users searching by username or by content, which will return a list. On the VIP page a grid of X accounts will be rendered and display a random tweet upon clicking the button. The project connects to a dotnet server which queries the X API, reshapes the data needed, and passes it back up to the frontend. Feel free to try it out live here.

Star Wars Explore

The Explore project developed with React JS queries the Star Wars API for all characters. The response data gets paginated neatly so all characters can be searched and found easily in the table. Initially the requests were coming in a blocked/synchronous format, but utilizing promises with async/await helped speed up the process of fetching/retrieving and lastly rendering the desired character data. Feel free to try it out live here.
